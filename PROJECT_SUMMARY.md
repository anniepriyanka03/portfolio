### Easiest Option: GitHub Pages
```bash
1. Create GitHub account
2. Create repository
3. Push files to GitHub
4. Enable Pages in settings
5. Live at: github.com/username/portfolio
```

### Recommended Option: Netlify
```bash

2. Connect GitHub

4. Get custom domain
5. Automatic HTTPS
```
